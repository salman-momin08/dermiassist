# **App Name**: SkinWise

## Core Features:

- User Authentication & Profile: User profile with settings, including account deletion and OAuth (Google) for signup/login.
- Dashboard: Dashboard displaying user activity, skin analysis history, and detailed AI analysis reports.
- AI Skin Analysis: AI-powered skin analysis: asks pre-analysis questions (medication, duration) and generates detailed report with do's and don'ts for skin cure.  Includes patient details and uploaded image.
- Report Generation: Option to download a detailed analysis report as a PDF, containing the user's details and the uploaded image.
- Analysis History: My Analyses page: History of skin analyses with options to delete analyses or download reports. Persists analyses between user sessions.
- Doctor Appointment: Find a Doctor: List of certified doctors with option to send appointment requests (online/offline). Ability to attach AI skin report to the request, for ease of doctor's understanding.
- AI Chatbot: FAQ chatbot available as a persistent floating button to assist users.

## Style Guidelines:

- Primary color: Deep blue (#3F51B5) to convey trust and expertise. Inspired by the medical field.
- Background color: Light blue (#E3F2FD), providing a clean and calming interface.
- Accent color: Light Purple (#7E57C2) for highlights and call-to-action buttons to convey innovation.
- Body and headline font: 'Inter', a sans-serif font providing a modern and clean aesthetic.
- Code font: 'Source Code Pro' for displaying code snippets, ensuring readability.
- Use a set of consistent, professional icons for easy navigation and quick comprehension.
- Implement a clean, responsive layout that adapts to different screen sizes, ensuring accessibility and optimal viewing experience across devices. Ensure consistent placement of the 'SkinWise' logo in the header.
